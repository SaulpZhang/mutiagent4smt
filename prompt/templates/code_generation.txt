## 生成步骤

### 第一步：分析
1. 验证指令的核心目标是什么？
2. IAM配置有几个Statement？每个Statement的Effect/Action/Principal/Condition？
3. 约束列表中每个约束对应什么逻辑？

### 第二步：编写Z3 Python代码
1. 导入：`from z3 import *; solver = Solver()`
2. 为每个字段声明变量并断言实际值
3. 定义辅助函数（操作符/键分类、条件兼容性等）
4. 定义验证逻辑（有效值集合、条件矛盾检测）
5. 组合整体有效性 → `solver.add(policy_valid)`
6. 调用 `execute_z3_python(code=...)` 工具

### 第三步：检查执行结果

- 失败 → 修正后重新执行
